# nowa-i18n

[![NPM version](https://img.shields.io/npm/v/nowa-i18n.svg?style=flat)](https://npmjs.org/package/nowa-i18n)

根据zh-cn.js自动生成翻译excel文档及生成en.js

---

## Feature

- 根据zh-cn.js自动生成翻译excel文档及生成en.js

## Install

```bash
$ npm i nowa -g
$ nowa install i18n
```

## Usage

```bash
$ nowa i18n --help
```
